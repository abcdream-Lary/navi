# ![logo](assets/icons/app_icon.ico) Navi - 导航应用

    一个简洁优雅的导航应用，基于 Flutter 开发，支持 Windows 和 Android 平台。

## ✨ 功能

- 📌 基础功能

  - 🔍 网站收藏管理
  - 📁 分类整理
  - 🚀 快速访问

- 📊 数据管理

  - 💾 导出、导入数据
  - 🗄️ 本地存储

- 🎨 界面样式
  - 📱 卡片布局
  - ⭕ 圆角设计
  - 💫 基础动画

## 🛠️ 开发工具

- 🔧 开发环境：
  - 🎯 Flutter
  - 📝 Visual Studio Code
  - 📱 Android Studio

- 🤖 AI 辅助：
  - 💻 [Cursor](https://cursor.sh/)
